# TerraProperty Frontend

Frontend application for **TerraProperty**, a real estate platform focused on premium property listings and investment opportunities across **Noida, Greater Noida, and Yamuna Expressway** regions.

This repository contains the complete client-side codebase responsible for delivering the public-facing website where users can explore properties, view project details, and submit inquiries.

---

## 🌐 About TerraProperty

TerraProperty is a real estate advisory and property discovery platform designed to help buyers and investors explore verified projects and make informed property decisions.

The platform focuses on:

- Premium residential and commercial projects
- Verified listings
- Professional property consultation
- Structured property discovery

---

## 🚀 Features

- Property listings and project showcase
- Detailed property pages with images and specifications
- Search and browsing interface for real estate opportunities
- Lead capture and inquiry forms
- Company information and service pages
- Responsive design for desktop, tablet, and mobile devices
- SEO-friendly page structure
